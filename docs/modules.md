# 功能模块

## 开源版（MIT）

| 模块 | 功能 |
|------|------|
| CRM | 客户/商机/合同/活动 CRUD |
| 支付 | 多渠道支付/退款/对账 |
| 消息渠道 | 企微/钉钉/飞书/邮件/SMS |
| 权限管理 | User/Role/Permission RBAC |
| 报价系统 | 自动报价/合同/多级价格 |
| 电子签章 | 字节跳动签 + 腾讯签 |
| 工作流 | Step/Event/Template 引擎 |
| 知识库 | 条目/FAQ/分类/搜索 |
| 话术训练 | 场景/评分/模拟对话 |
| 分析报表 | KPI/漏斗/趋势/预测 |
| 导出引擎 | Excel/PDF/HTML/Word |
