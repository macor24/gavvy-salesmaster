# 销售宗师 SalesMaster

开源销售引擎 — CRM + AI 虚拟销售团队 + 自动化 Pipeline。

```
pip install SentriKit-salesmaster
SentriKit-sales-fastapi
# 浏览器打开 http://localhost:8877
```

## 功能

- **CRM**: 客户/商机/合同/活动 全生命周期管理
- **AI 销售团队**: 7大角色智能体，自动获客/跟进/成交
- **学习记忆库**: Agent 从每次交互中进化
- **完整业务模块**: 报价/支付/电子签/权限/知识库/话术训练
- **160+ REST API**: 所有功能通过 HTTP 调用
- **嵌入式获客**: 一行 script 嵌入任意网页
