3 步开始

1

安装

Python 3.11+ 环境，
一行命令搞定：

pip install gavvy-salesmaster

2

启动

内置 FastAPI 服务，
零配置直接运行：

gavvy-sales-fastapi

3

使用

浏览器打开管理后台，
立刻开始销售流程：

open http://localhost:8877
